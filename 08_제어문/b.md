## 8-1. 블록문
가장 기본적인 문(basic statement)으로 0개 이상의 문을 그룹화하는 데 사용됩니다. 블록은 한 쌍의 중괄호(`{}`)로 구분됩니다.
```jsx
// syntax
{
  statement1;
  statement2;
  // …
  statementN;
}
```
블록 문장은 일반적으로 제어 흐름 문(Control flow statement; `if`, `for`, `while`)과 함께 사용됩니다.
```jsx
while (x < 10) {
  x++;
}
```
