# 4장. 변수

- **변수(variable)**
  변수 : 하나의 값을 저장하기위해 확보한 메모리 공간 자체 또는 그 메모리 공간을 식별하기 위해 붙인 이름
  여러값을 저장하려면 여러개의 변수를 사용해야한다. 단, 배열이나 객체같은 자료구조를 사용하면 관련이 있는 여러개의 값을 그룹화해서 하나의 값처럼 사용할 수 있다. 
  
  <br>
  
  ```javascript
  var userId = 1;
  var userName = "choi";
  // userId는 변수 이름, 1은 변수의 값
  var user = {id:1,name:'choi'};
  var users = [
    {id:1,name:'choid'},
    {id:2,name:'lee'}
  ]
